函数(function)

1.简介

A function is a block of code which only runs when it is called.

You can pass data, known as parameters, into a function.

A function can return data as a result.

你可以定义一个由自己想要功能的函数，以下是简单的规则：

  • 函数代码块以 def 关键词开头，后接函数标识符名称和圆括号()。
  • 函数名必须要符号标识符的规范（可以包含字母、数字、下划线、但是不能以数字开头）
  • 任何传入参数和自变量必须放在圆括号中间。圆括号之间可以用于定义参数。
  • 函数的第一行语句可以选择性地使用文档字符串—用于存放函数说明。
  • 函数内容以冒号起始，并且缩进。
  • return [表达式] 结束函数，选择性地返回一个值给调用方。不带表达式的return相当于返回 None。

3.定义一个函数

def my_function():
    print("Hello from a function")

4.函数的调用

my_function()

5.参数

From a function's perspective:

A parameter is the variable listed inside the parentheses in the function definition.

An argument is the value that are sent to the function when it is called.
  • 在定义函数时，可以在函数名后的()中定义数量不等的形参，
    多个形参之间使用,隔开
  • 形参（形式参数）(parameter)
    • 定义形参就相当于在函数内部声明了变量，但是并不赋值
  • 实参（实际参数）(argument)
    • 如果函数定义时，指定了形参，那么在调用函数时也必须传递实参，
      实参将会赋值给对应的形参，简单来说，有几个形参就得传几个实参
1.必需参数(位置参数)
def my_function(fname, lname):
    print(fname + " " + lname)

my_function("Emil", "Refsnes")
  • 必需参数须以正确的顺序传入函数。调用时的数量必须和声明时的一样。
2.不定长参数(*args)
def my_function(*kids):
    print("The youngest child is " + kids[2])

my_function("Emil", "Tobias", "Linus")
  • 不定长参数可以一次传入过个参数,传入参数会生成一个元组(tuple).
3.关键字参数
def my_function(child3, child2, child1):
    print("The youngest child is " + child3 + child2 + child1)

my_function(child1 = "Emil", child2 = "Tobias", child3 = "Linus")
  • 关键字参数可以直接给位置参数赋值,可以不按找位置参数顺便.
4.任意关键字参数(**kwargs)

kwargs(keyword arguments)

def my_function(**kid):
    print("His last name is " + kid["lname"])

my_function(fname = "Tobias", lname = "Refsnes")
  • 任意关键字参数可以传入多个关键字参数,传入参数会生成一个字典(dictionary).
5.默认值参数
def my_function(country = "Norway"):
    print("I am from " + country)

my_function("Sweden")
my_function("India")
my_function()
my_function("Brazil")
  • 默认值参数在函数声明的时候已经赋值,可以不传参数.
6.混合参数
def my_function(name, *hobby, sex = 'male', **family):
    print(f"I am {name} {sex} {hobby} {family}")
my_function("Jivan", "sing", "movie", father = "dad", mother = "mom")
  • 声明函数是参数的顺序:
    • 位置参数 > *args > 默认值参数 > **kwargs
7.不定长参数和任意关键字参数传参
def func(*args,**kwargs):
    return args , kwargs
func(*[1,2,3,4,5], **{'a':'1','b':'2'})
lis = [6,7,8,9,10]
dic = {'c':'3','d':'4'}
func(*lis, **dic)
  • 不定长参数传参的时候可以在可迭代对象前面加*号.
  • 任意关键字参数传参的时候可以在字典前面加**.
